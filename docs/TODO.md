# Development Roadmap: Pixel Chess Roguelike

> 모드 구조에 맞춘 마일스톤 기반 로드맵. 각 마일스톤은 완료 시 의미 있는 사용자 가치를 제공하도록 구성.

## M0: 공통 기반 (Foundation)

프로젝트 셋업과 모드 진입점 골격.

- [ ] Bun 프로젝트 초기화 (`bun init`)
- [ ] Frontend: SolidJS + Vite + Tailwind CSS 설정
- [ ] Backend: Hono 기본 서버 구축 (옵션, 서버 기능 미사용 시 보류 가능)
- [ ] Phaser 3 초기화 및 SolidJS 컴포넌트 연동용 Wrapper 작성
- [ ] `chess.js` 통합 및 기본 보드 렌더링
- [ ] 기본 체스보드(8x8) 타일셋 및 기물 에셋(Placeholder) 준비
- [ ] 메인 메뉴 → 클래식/모험 카드 라우팅 골격
- [ ] 글로벌 설정(사운드/언어/테마) 저장 구조

## M1: 클래식 공통 + 싱글 (Classic Common + Single)

vs AI 대전을 끝까지 플레이할 수 있는 상태.

### 클래식 공통
- [ ] 시간 제어 시스템 (Bullet/Blitz/Rapid/Classical/무제한/커스텀)
- [ ] 종료 조건 처리 (체크메이트/스테일메이트/50수/3회 동형/시간 만료/기권)
- [ ] 보드 UI 공통 (좌표·합법 수·라스트 무브·체크 강조)
- [ ] 폰 승급 다이얼로그
- [ ] 기보 표기(SAN) 및 PGN 내보내기

### 싱글
- [ ] Stockfish.js Web Worker 통합 (UCI 프로토콜)
- [ ] 난이도 5단계 프리셋 (UCI_Elo + Skill Level 매핑)
- [ ] 커스텀 옵션 (ELO/사고시간/Contempt 슬라이더)
- [ ] 힌트 기능 (강·중·강 + 횟수 제한)
- [ ] 무르기 기능 (옵션, 횟수 제한)
- [ ] 분석 모드 (평가바, MultiPV, 수 되감기)
- [ ] 게임 히스토리 자동 저장 (IndexedDB)

## M2: 클래식 로컬멀티 (Classic Local Multiplayer)

2인 핫시트 대국 완성.

- [ ] 보드 자동 회전 (옵션 ON/OFF, 200ms 트랜지션)
- [ ] 무르기 합의 모달 (요청·동의·거부)
- [ ] 무승부 제안 모달 (요청·동의·거부)
- [ ] 기권 확인 모달
- [ ] 상하 시계 위젯 + 차례 인디케이터
- [ ] Wake Lock (화면 자동 잠금 방지)
- [ ] 우발적 입력 방지 (드래그/두 단계 탭만 허용)
- [ ] 다시 두기 시 색 자동 교대

## M3: 모험모드 MVP (Adventure MVP)

기본 캐릭터 1종으로 1막 클리어까지 가능한 상태.

### 시스템
- [ ] 모험 모드 진입 흐름 (캐릭터 선택 → 맵)
- [ ] 노드 맵 생성 알고리즘 (Battle/Elite/Shop/Event/Rest/Boss + 가중치)
- [ ] 맵 화면 UI (노드 그래프, 현재 위치, 진입 가능 표시)
- [ ] HP/ATK 스탯 시스템 (기물별)
- [ ] 전투 처리 로직 (캡처 시 데미지 계산, 잔존 HP 처리)
- [ ] 앙파상/승급 엣지 케이스 처리
- [ ] 스테일메이트 → 패배 처리
- [ ] 기본 아이템 풀 (Common 10종, Uncommon 5종 정도)
- [ ] 아이템 슬롯 시스템 (기물별 2슬롯)
- [ ] 인벤토리 UI

### 1막 콘텐츠
- [ ] 기본 캐릭터(Standard) 1종
- [ ] 1막 보스 1종 (다단계 페이즈)
- [ ] Battle/Elite/Shop/Event/Rest 노드 콘텐츠 (각 종류 최소 1개 변형)
- [ ] 런 시작 → 1막 보스 클리어 → 런 결과 화면

## M4: 메타 진행 + 추가 콘텐츠 (Meta Progression + Content)

영구사망 + 메타 진행 + 추가 막/캐릭터.

### 메타 진행
- [ ] 별의 조각 시스템 (적립 규칙, 영구 저장)
- [ ] 메타 진행 화면 (해금 트리)
- [ ] 영구 장식품 (시작 골드 +N, 시작 HP 보너스 등)
- [ ] 자동 이어하기 (앱 종료 후 진행 중 런 복구)

### 캐릭터 해금
- [ ] 암살자단 (나이트 강화 파티)
- [ ] 신성단 (비숍·킹 강화 파티)
- [ ] (선택) 요새단·혼돈단

### 추가 막
- [ ] 2막 (8노드 + 보스)
- [ ] 3막 (8노드 + 최종 보스)

### 콘텐츠 확장
- [ ] 아이템 풀 확장 (Rare 5종 + Legendary 2종 추가)
- [ ] 글로벌 모디파이어 시스템
- [ ] 이벤트 노드 콘텐츠 확장 (분기 텍스트)

## M5: 폴리시 (Polish)

정식 출시 품질로 끌어올리는 단계.

### 비주얼
- [ ] 정식 도트 에셋 적용 (기물·배경·이펙트)
- [ ] 노드 아이콘·보스 스프라이트·캐릭터 초상화 정식 에셋
- [ ] 아이템 카드 등급별 프레임 정식 에셋
- [ ] 기물 애니메이션 (Phaser Tween: Idle/Move/Capture)

### 오디오
- [ ] BGM (메인메뉴/클래식/막별/보스/결과)
- [ ] SFX 전체 (글로벌/클래식 공통/모험 전용)
- [ ] 사운드 볼륨 설정 분리 (BGM/SFX)

### 도전과제 / 랭킹 (옵션)
- [ ] 모험 도전과제 시스템
- [ ] 서버 랭킹 (옵션, Hono API + SQLite)
- [ ] 사용자 인증 (선택 사항: 간단한 닉네임 기반)

### 최적화 / 접근성
- [ ] 모바일 웹 브라우저 최적화 (Phaser Scale FIT, Tailwind Responsive)
- [ ] 모션 감소 옵션 (접근성)
- [ ] 한국어/영어 i18n
- [ ] 배포 (Vercel/Fly.io 등)
