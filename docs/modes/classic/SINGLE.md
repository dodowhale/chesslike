# Classic Mode — Single (vs AI)

> 클래식 서브모드 중 하나. 사용자 vs Stockfish AI 대국.
> 정규 체스 룰·시간 제어·종료 조건 등 공통 사양은 [COMMON.md](./COMMON.md) 참조. 본 문서는 싱글모드에만 해당하는 사항만 다룬다.

## 1. 모드 목적

- 정통 체스를 학습·연습·도전한다.
- 다양한 ELO의 AI 상대로 자기 수준에 맞는 게임을 한다.
- 분석·힌트 기능으로 학습을 보조한다.

## 2. AI 난이도 시스템

### 2.1 프리셋 5단계

| 프리셋 | 대략 ELO | 대상 |
|---|---|---|
| 초보 (Novice) | 800 | 룰만 아는 초심자 |
| 아마추어 (Amateur) | 1200 | 동네 대국 수준 |
| 중급 (Intermediate) | 1600 | 동호회 수준 |
| 고급 (Advanced) | 2000 | 토너먼트 수준 |
| 마스터 (Master) | 2400+ | 최고 난이도 |

### 2.2 커스텀 옵션

프리셋 외에 직접 조절 가능:

| 옵션 | 범위 | 기본값 | 설명 |
|---|---|---|---|
| ELO | 600~3000 | 1200 | UCI_Elo로 매핑 |
| 사고 시간 | 100~5000ms | 1000 | 한 수당 최대 사고 시간 |
| Contempt | -100~+100 | 0 | 무승부 회피 성향 (+면 무승부 싫어함) |

커스텀 모드 진입 시 위 슬라이더가 노출된다.

### 2.3 Stockfish UCI 옵션 매핑

| 프리셋 | UCI_Elo | Skill Level | Depth 제한 | 사고 시간 |
|---|---|---|---|---|
| 초보 | 800 | 3 | 5 | 500ms |
| 아마추어 | 1200 | 8 | 10 | 700ms |
| 중급 | 1600 | 13 | 15 | 1000ms |
| 고급 | 2000 | 18 | 20 | 2000ms |
| 마스터 | 2400+ | 20 | 무제한 | 3000ms |
| 커스텀 | 사용자 설정 | 자동 매핑 | — | 사용자 설정 |

공통 옵션: `Threads=1`, `Hash=64MB` (모바일 친화). `MultiPV=1` (분석 모드에서만 3).

## 3. 보조 기능

### 3.1 힌트

- **약한 수**: AI가 4th best 정도의 수를 제시 (학습 자극용)
- **평균 수**: 2nd best
- **최선 수**: best move (분석)

설정에서 사용 가능 여부와 횟수 제한을 정한다 (기본: 게임당 3회).

### 3.2 분석 모드

- 게임 종료 후 또는 일시정지 시 진입.
- **평가바**: 좌측 세로 바로 현재 평가치(centipawn 또는 mate-in-N) 표시.
- **MultiPV 변동수**: 상위 3개 후보 수와 평가치 표시.
- **수 되감기/되돌리기**: 임의 수로 이동 가능.

### 3.3 무르기

- 옵션 ON 시 가능. 기본 OFF.
- 횟수 제한 설정 (1~10회 또는 무제한).
- 무르기 시 AI 응답도 함께 되돌림.

### 3.4 기보 저장

- 자동: 모든 게임은 히스토리에 자동 저장.
- 수동 내보내기: PGN 파일 다운로드.
- PGN 헤더: 모드(Classic-Single), AI 난이도, 시간 제어, 결과.

## 4. UI 와이어 개요

```
┌─────────────────────────────────────────────────────────┐
│                  [Pause]   [Settings]                    │
├──────────────────────────────┬──────────────────────────┤
│                              │  평가바                   │
│                              │  ┃                       │
│         보드 (8x8)            │  ┃                       │
│                              ├──────────────────────────┤
│                              │  기보                    │
│                              │  1. e4 e5                │
│                              │  2. Nf3 Nc6              │
│                              ├──────────────────────────┤
│                              │  [힌트] [무르기] [기권]    │
└──────────────────────────────┴──────────────────────────┘
```

- 모바일: 사이드 패널이 하단 드로어로 접힘. 보드는 가로 폭 100%.
- 데스크탑: 사이드 패널 고정 표시.
